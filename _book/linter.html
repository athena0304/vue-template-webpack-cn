Linter Configuration

Linter 配置

This boilerplate uses ESLint as the linter, and uses the Standard preset with some small customizations.

该模板使用ESLint作为提示检查器，并且使用标准的 预设和一些小的定制。

如果你不满意默认的检验规则，有下面几个选择：

  1. 重写.eslintrc.js里的自定义规则。例如，你可以添加下列规则来强制使用分号，而不是忽略分号。

    // .eslintrc.js
"semi": [2, "always"]

  2. 在生成项目的时候选择一个不同的ESLint预设，例如eslint-config-airbnb.

  3. 生成项目的时候不做ESLint预设，然后再顶你你自己的规则。查看 ESLint 文档 获取更多细节。
