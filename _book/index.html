概述

该模板是针对大型的严格的工程，并且要求你或多或少岁webpack和vue-loader有一定的了解。一定要也看一下vue-loader's documentation 来了解一下普通的工作流方法。

如果你只想尝试一下vue-loader或者快速开始，那么用webpack-simple这个就好。

快速开始

这个模板是基于vue-cli这个脚手架搭建的。为了确保使用更有效的依赖树，强烈建议使用npm 3以上的版本。

$ npm install -g vue-cli
$ vue init webpack my-project
$ cd my-project
$ npm install
$ npm run dev
